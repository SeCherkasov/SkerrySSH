功能特性

一个客户端，覆盖所有运维场景

本地优先，零知识，主流平台功能对等。

全平台一致体验
Linux / Windows / macOS / Android，同一套核心与 UI，功能完全对等——一个特性没全平台跑通就不算完成。
本地优先
无需注册、无需服务器，开箱即用。所有数据默认留在本机，账号与云同步只是可选项。
加密保险库
Argon2id + XChaCha20-Poly1305 始终加密，主密码永不离设备，零知识设计。
自托管零知识同步
同步服务器可自行部署，服务端只见密文。也可直连官方同步实例，无需运维。
三个密码，各管各的
主密码锁本机 · 同步账户密码连服务器 · 配对密码加设备。三者独立，忘了一个不牵连其他。
团队共享
主机、密钥与配置可端到端加密共享给团队成员，权限粒度到单个连接。
终端 & 双栏 SFTP
内置终端模拟器，多标签会话管理。双栏可视化 SFTP，拖拽传文件。
端口转发
本地 / 远程 / 动态（SOCKS）三种转发模式全支持，隧道随连接自动建立。
Serial / Telnet / Mosh
除 SSH 外还支持串口与 Telnet 直连；Mosh 让移动网络下的会话掉线不断。
AI 辅助（可控）
本地模型或自带 Key 的云端模型均可选。模型输出视为不可信，每条命令执行前需你确认。
完全开源
客户端 GPL-3.0、服务端 AGPL-3.0，代码完全开放，自由审计与再分发。
