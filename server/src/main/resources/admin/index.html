Two passwords, two jobs
Device password (master password)
Lives only on this device and unlocks your local vault. Each device can have its own.
Sync password
Set when first enabling sync — identifies your account on the server and encrypts cloud data. Must be the same on every device, or synced data can't be decrypted.
In short: the device password unlocks this machine; the sync password unlocks the cloud. The server never sees either.
